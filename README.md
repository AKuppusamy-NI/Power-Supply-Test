# Power-Supply-Test

This repository is a simple demo for test engineers collaborating on power supply test workflows.

It represents a TestStand-style development flow with a readable sequence file, a small Python measurement helper, and a basic configuration file.

Engineers can update these files, review changes, and sync their work through GitHub as they iterate on test sequences together.

## Repository Contents

- `test_sequence.seq` - demo main test sequence showing the workflow steps
- `measure_voltage.py` - simple Python script that simulates a voltage measurement
- `config.json` - demo configuration values such as target voltage and tolerance

## Purpose

This repository is intended for workflow collaboration demos, not production test execution.
